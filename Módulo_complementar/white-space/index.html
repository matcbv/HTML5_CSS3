Exemplos de White-Space

white-space: normal;
Este é um texto de exemplo. Os espaços e quebras de linha serão colapsados de forma padrão.
white-space: nowrap;
Este é um texto de exemplo. Os espaços são colapsados, mas não haverá quebras de linha automáticas.
white-space: pre;
Este é um texto de exemplo. Os espaços e quebras de linha serão preservados.
white-space: pre-line;
Este é um texto de exemplo. Os espaços são colapsados, mas as quebras de linha serão preservadas.
white-space: pre-wrap;
Este é um texto de exemplo. Os espaços e quebras de linha serão preservados, mas o texto será quebrado quando necessário para se ajustar ao contêiner.
